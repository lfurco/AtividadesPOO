# 📚 Atividades POO - Java

Repositório com exercícios práticos de Programação Orientada a Objetos (POO) em Java.

## 🔹 Conteúdo

As atividades abordam conceitos fundamentais de POO:

- ✔ Classes e Objetos
- ✔ Encapsulamento
- ✔ Associação
- ✔ Agregação
- ✔ Composição
- ✔ Uso de ArrayList
- ✔ Relacionamento entre classes

---

## 📂 Estrutura do Projeto

- `AtividadeClinica` → Sistema de Clínica Médica  
- `AtividadeBiblioteca` → Sistema de Biblioteca  
- `AtividadeBasquete` → Time de Basquete  
- `AtividadeSoftware` → Projeto de Software  
- `AtividadeComputador` → Computador e Componentes  
- `AtividadeApartamentos` → Edifício e Apartamentos  
- `AtividadeVendas` → Sistema de Vendas  
- `AtividadeCineTeatro` → Sistema de Cine-Teatro
